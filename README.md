# 🧠 Legacy AI Vault

> Built at **Curserxthinc! Hackathon** — March 2026

## What is this?

Legacy AI Vault is a project born at the Curserxthinc! Hackathon. The idea: preserve, organize, and bring back to life forgotten AI experiments, old prompts, and legacy model outputs — all in one place.

## Features

- 🗄️ Archive old AI conversations and outputs
- 🔍 Search and rediscover past experiments
- 🤖 Revive legacy prompts with modern models
- 📂 Organized vault structure for AI artifacts

## Getting Started

```bash
git clone https://github.com/Luraxx/legacy_ai_vault.git
cd legacy_ai_vault
```
